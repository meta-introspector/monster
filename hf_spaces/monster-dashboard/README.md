# Monster Review Dashboard

Real-time dashboard for the 9-persona Monster review team tracking code quality, formal proofs, and dataset generation.

## Features

- **9-Persona Reviews**: Knuth, ITIL, ISO9001, GMP, Six Sigma, Rust Enforcer, Fake Detector, Security Auditor, Math Professor
- **Formal Proofs**: Lean4 theorem verification
- **ZK-ML Proofs**: Zero-knowledge performance proofs
- **HuggingFace Datasets**: Automatic parquet uploads
- **GitHub Actions**: CI/CD integration

## Metrics

- Review scores (0-90)
- Proof verification status
- Dataset upload tracking
- Performance benchmarks

## Data Sources

- GitHub Actions artifacts
- HuggingFace datasets: `introspector/data-moonshine`, `meta-introspector/monster-perf-proofs`
- Git commit metadata with RDFa proofs
